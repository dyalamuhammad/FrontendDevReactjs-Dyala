Website dengan data data restaurant dari API terbuka yang saya buat. Data restaurant mencakup; nama, harga, rating, keterangan buka/tutup

## Versi React.js
18.2.0

## Versi Node.js
18.17.1

### Menggunakan npm
```bash
# Install semua dependensi
npm install

# Menjalankan proyek secara lokal
